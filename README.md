# Credit Card Transaction Analysis

Analyze credit card transaction data to uncover spending patterns, detect trends, and provide actionable insights for financial decision-making using Python, Pandas, NumPy, and visualization tools.

*Tools:* Python, Pandas, NumPy, Power bi, Excel.

*Key Features:*
- Transaction trend analysis over time
- Categorization of spending by type and merchant
- Identification of high-value transactions
- Summary dashboards for insights
